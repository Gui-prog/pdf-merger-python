# PDF Merger em Python 

Script simples para mesclar múltiplos arquivos PDF em um único arquivo.

## Funcionalidades 
- Junta vários PDFs em um só
- Ordena automaticamente os arquivos 
- Fácil de usar

## Tecnologias
- Python 
- PyPDF2

## Como usar

1. Coloque seus arquivos PDF na pasta `arquivos`
2. Execute o script:

```bash
python projeto1.py
